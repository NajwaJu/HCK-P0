# 🏠 Bantu Ibu Kos

Web app sederhana untuk membantu pemilik kos mengelola kamar, penghuni, dan pembayaran.

Project ini dibuat untuk **Hacktiv8 Hackathon Phase 0** menggunakan:
- HTML
- CSS / Bootstrap
- Vanilla JavaScript (tanpa framework)

## 🎯 Project Background

Banyak pemilik kos di Indonesia masih mencatat:
- Kamar kosong / terisi
- Pembayaran bulanan
- Data penghuni

secara manual (buku / Excel).

Aplikasi ini bertujuan membantu pemilik kos mengelola bisnis kos secara sederhana melalui web app yang terstruktur dan mudah digunakan.

---

## ✨ Main Features (MVP)

### 📊 Dashboard
Menampilkan ringkasan:
- Total kamar
- Jumlah kamar kosong
- Jumlah kamar terisi
- Jumlah kamar belum bayar

---

### 🚪 Room Management (CRUD)

Fitur:
- Create → Tambah kamar
- Read → Lihat daftar kamar
- Update → Edit harga & tipe kamar
- Delete → Hapus kamar

Data kamar:
- Nomor kamar
- Harga sewa
- Tipe kamar
- Status (Kosong / Terisi)

---

### 👤 Tenant Management (CRUD)

Fitur:
- Check-in penghuni baru
- Daftar penghuni
- Pindah kamar
- Check-out penghuni

Data penghuni:
- Nama
- Nomor HP
- Nomor kamar
- Tanggal masuk

---

### 💰 Payment Management (CRUD)

Fitur:
- Tambah pembayaran
- Lihat tagihan
- Update status pembayaran
- Hapus data pembayaran

Tipe pembayaran:
- Sewa bulanan
- Sewa + deposit saat check-in
